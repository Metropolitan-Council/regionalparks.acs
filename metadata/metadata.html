Section 1: Overview

Originator:Metropolitan Council

Title: Annual Small Area Population and Household Estimates, Twin Cities Metropolitan Area

Abstract: This dataset consists of housing unit, household, and population estimates for census tracts, census block groups, Transportation Analysis Zones (TAZs), school districts, and ZIP codes in the Twin Cities Region. These data provide a more precise and timely picture of current conditions than the American Community Survey, another source of small area data. It may be possible to calculate estimates for other small areas upon request; contact Research@metc.state.mn.us for more information.

Purpose: This dataset consists of housing unit, household, and population estimates for census tracts, census block groups, Transportation Analysis Zones (TAZs), school districts, and ZIP codes in the Twin Cities Region. Spatial Extent of Data: Twin Cities seven-county metropolitan area. This includes the counties of Anoka, Carver, Dakota, Hennepin, Ramsey, Scott and Washington in Minnesota. Estimates for Transportation Analysis Zones also include counties that are contiguous to the region (Chisago, Goodhue, Isanti, Le Sueur, McLeod, Rice, Sherburne, Sibley, and Wright Counties in Minnesota and Pierce, Polk, and St. Croix Counties in Wisconsin).

Section 2: Data Quality

Attribute Accuracy: By definition, estimates are only approximations of the characteristics of an area at a point in time. These estimates are based on the best available information provided by local governments, U.S. Census Bureau, and U.S. Postal Service. They are also designed to be consistent with city and township estimates from the Metropolitan Council and, for TAZs in "collar counties," estimates from the Minnesota State Demographic Center and Wisconsin Demographic Services Center (TAZs in "collar counties" only). See the estimates methodology, available via metrocouncilorg/populationestimates, for an explanation of the calculations.

Logical Consistency:

Completeness: Entire seven-county geographic area is covered. Transportation Analysis Zone estimates also cover "collar counties" (those contiguous to the seven-county region).

Section 5: Attributes

Overview: STATE: FIPS code for state (Minnesota = 27; Wisconsin = 55)
CO_CODE: FIPS code for county, available for tract/block group/TAZ estimates only
TR10: FIPS code for census tracts, available for tract estimates only (digits 1-2 = state; digits 3-5 = county; digits 6-11 = tract)
BG10: FIPS code for census block groups, available for block group estimates only (digits 1-2 = state; digits 3-5 = county; digits 6-11 = tract; digit 12 = block group)
TAZ_2010: Code for Transportation Analysis Zone (TAZ), available for TAZ estimates only
UNI_TYP: Type of school district, available for school district estimates only (1 = independent; 3 = special)
SDNUM: School district identifier, available for school district estimates only
SDNAME: School district name, available for school district estimates only
ZCTA5: ZIP Code Tabulation Area identifier, available for ZCTA estimates only
PARTIALGEO: Indicator for whether the estimate represents the complete geography or only part of it, available for school district and ZCTA estimates only (1 = geography lies partly outside the seven-county Twin Cities region, and estimate is only for the portion inside the region)
EST_YEAR: Year of current estimate (2019)
HU_EST: Total housing units, current estimate
HH_EST: Total households, current estimate
POP_EST: Total population, current estimate
POPHH_EST: Population in households, current estimate
POPGQ_EST: Population in group quarters, current estimate
